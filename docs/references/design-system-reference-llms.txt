GoGo Design System — AI Quick Reference
========================================
Source: frontend/src/app/globals.css + frontend/DESIGN_SYSTEM.md
Last updated: 2026-04-04

RULE #1: 하드코딩 hex/rgba 절대 금지. 아래 토큰만 사용.
RULE #2: dark: 접두사 사용 금지. CSS 변수가 자동으로 다크모드 처리.
RULE #3: 카테고리 그라디언트(CATEGORY_GRADIENT)는 예외 — 의도적 hex arbitrary value.


COLOR TOKENS
============

배경 (Background)
-----------------
bg-bg              페이지/모달 배경         light=#FFFFFF     dark=#1A1726
bg-bg-secondary    카드/섹션 배경           light=#F6F4FB     dark=#221F30
bg-surface         입력필드/탭 비활성 배경   light=#F0EDF8     dark=#2A2640
bg-surface-hover   surface hover 상태      light=#E8E4F3     dark=#353050

텍스트 (Text)
-------------
text-text-main       제목, 본문              light=#2D264B     dark=#EDEAF5
text-text-secondary  보조 텍스트             light=#5C5478     dark=#B5B0C8
text-text-muted      placeholder, 부가 정보  light=#736C8E     dark=#908AAC
text-text-on-primary primary 배경 위 텍스트  light=#FFFFFF     dark=#1A1726

포인트 컬러 (Primary = Lavender)
---------------------------------
bg-primary         CTA 버튼, 활성 탭, 링크   light=#9D8DC2     dark=#B0A0D8
bg-primary-hover   primary hover 상태       light=#8A79B4     dark=#C4B8E6
bg-primary-subtle  뱃지/태그 배경, 선택 배경  light=#EDE9F6     dark=#2E2A42
text-primary       아이콘 강조, 링크 텍스트   same as bg-primary

악센트 (Accent = Pink)
-----------------------
text-accent        좋아요 하트, 강조 텍스트   light=#FFB5C5     dark=#FF9DB2
bg-accent-bg       좋아요 활성 배경           light=#FFF0F4     dark=#2D1F28
text-accent-text   좋아요 활성 텍스트         light=#D4436A     dark=#FFB5C5

위험/삭제 (Danger)
-------------------
text-danger        삭제 버튼, 에러 메시지     light=#E05178     dark=#FF7A9A
bg-danger-bg       삭제 hover 배경           light=#FFF0F4     dark=#2D1F28

성공 (Success)
--------------
text-success       성공 상태, 완료 표시       light=#6BBF7B     dark=#7DD88C
bg-success-bg      성공 배경                 light=#F0FAF2     dark=#1C2D1F

테두리 (Border)
----------------
border-border      기본 테두리               light=#E8E4F0     dark=#363150
border-border-hover hover 테두리             light=#D4CDE4     dark=#4A4468


BUTTON PATTERNS
===============

Primary CTA (가장 중요한 액션)
  className="bg-primary hover:bg-primary-hover text-text-on-primary rounded-xl px-4 py-2"

Secondary (폼 제출, 확인)
  className="bg-text-main hover:bg-text-secondary text-text-on-primary rounded-xl px-4 py-2"

Ghost / Cancel
  className="bg-surface hover:bg-surface-hover text-text-secondary rounded-xl px-4 py-2"

Danger (삭제)
  className="text-danger hover:bg-danger-bg rounded-xl px-4 py-2"

Disabled
  className="... opacity-50 cursor-not-allowed"


CARD / SURFACE PATTERNS
========================

기본 카드
  className="bg-bg-secondary rounded-2xl p-4 border border-border"

인터랙티브 카드 (클릭 가능)
  className="bg-bg-secondary hover:bg-surface rounded-2xl p-4 border border-border hover:border-border-hover transition-colors cursor-pointer"

모달 / 오버레이
  className="bg-bg rounded-2xl p-6 shadow-lg"


INPUT PATTERNS
==============

기본 입력 필드
  className="bg-surface border border-border rounded-xl px-4 py-2 text-text-main placeholder:text-text-muted focus:outline-none focus:ring-2 focus:ring-primary"

포커스 스타일 (전역 적용됨)
  :focus-visible { outline: 2px solid var(--color-primary); outline-offset: 2px; border-radius: 4px; }
  → 별도 focus 스타일 추가 불필요


TYPOGRAPHY
==========

폰트: Pretendard Variable (globals.css에서 import)
letter-spacing: -0.02em (body 전역 적용)

제목 계층
  h1: text-2xl font-bold text-text-main
  h2: text-xl font-semibold text-text-main
  h3: text-lg font-medium text-text-main
  부제목: text-sm text-text-secondary


ANIMATION
=========

전역 transition (globals.css에서 자동 적용)
  color, background-color, border-color, box-shadow, transform — 200ms

prefers-reduced-motion 대응 (자동 적용됨)
  → 별도 처리 불필요

커스텀 애니메이션 추가 시
  className="transition-[원하는속성] duration-200"


ACCESSIBILITY
=============

화면 독자용 텍스트
  className="sr-only"
  → globals.css에 정의됨

포커스 인디케이터
  :focus-visible 전역 적용됨
  → tabIndex={0} + role 추가 시 자동으로 포커스 링 표시

ARIA 패턴
  aria-label="버튼 설명"           → 아이콘만 있는 버튼
  role="status" aria-live="polite" → 동적으로 바뀌는 상태 텍스트
  aria-expanded={isOpen}           → 드롭다운/아코디언


DARK MODE
=========

구현 방식: class 전략 — <html class="dark"> 토글
토글 위치: ThemeContext (frontend/src/lib/ThemeContext.tsx 또는 유사 경로)

AI 작업 시 주의사항
  - dark: 조건부 클래스 추가 불필요 — CSS 변수가 자동 처리
  - bg-bg, text-text-main 등 토큰 쓰면 다크모드 자동 대응
  - 예외: 이미지 overlay, 특수 gradient는 별도 처리 필요할 수 있음


LEGACY ALIASES (사용 자제)
==========================

--color-mint      → bg-primary 사용
--color-green     → text-text-main 사용
--color-green-mid → text-text-secondary 사용
--color-card-bg   → bg-surface 사용


WHAT NOT TO DO
==============

❌ className="text-[#9D8DC2]"          → ✅ className="text-primary"
❌ className="bg-white dark:bg-[#1A1726]" → ✅ className="bg-bg"
❌ style={{ color: '#736C8E' }}         → ✅ className="text-text-muted"
❌ className="dark:text-white"          → ✅ className="text-text-main"
